# Sudoku Reveal Logo Assets

Generated from `scripts/generate-logo-assets.mjs`.

The logo mark uses an accurate 9 x 9 Sudoku board with heavier 3 x 3
subdivision lines and lifted cells revealing artwork underneath.

## Primary App Use

- `sudoku-reveal-home-wordmark-tight.png` - main screen on light/image backgrounds.
- `sudoku-reveal-home-wordmark-no-lines.png` - no-lines variant with a wider canvas.
- `sudoku-reveal-home-wordmark.png` - earlier home-screen wordmark variant.
- `sudoku-reveal-horizontal-transparent-darktext.png` - alternate transparent horizontal lockup.
- `sudoku-reveal-horizontal-transparent.png` - dark surfaces or image backgrounds with dark overlays.
- `sudoku-reveal-app-icon.png` - app icon source composition.

## Full-Color Variants

- `sudoku-reveal-horizontal-dark.png` - horizontal lockup on black background.
- `sudoku-reveal-horizontal-light.png` - horizontal lockup on white background.
- `sudoku-reveal-square-dark.png` - square lockup on black background.
- `sudoku-reveal-square-light.png` - square lockup on white background.
- `sudoku-reveal-mark.png` - standalone icon mark.

## Single-Color Variants

- `sudoku-reveal-mono-black.png` - black single-color horizontal lockup.
- `sudoku-reveal-mono-white.png` - white single-color horizontal lockup on black.

Each PNG has a matching SVG source with the same base filename.
